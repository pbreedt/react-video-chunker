# react-video-chunker
> A POC for chunking video during upload using React

## Technologies
* React - 18.3.1
* Node

## Run

```bash
# Server
$ cd server
$ npm install
$ node server.js

# Client
$ cd client
$ npm install
$ npm run start
```
